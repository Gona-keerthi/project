Tech glosary

HTML
HTML (HyperText Markup Language) is the standard language used to create and structure web pages by defining elements such as text, images, links, and multimedia using tags. It forms the basic foundation of web development and works together with CSS for styling and JavaScript for interactivity. HTML helps browsers understand and display content correctly, making websites organized and accessible.
CSS
CSS (Cascading Style Sheets) is used to style and design web pages by controlling layout, colors, fonts, spacing, and responsiveness. It separates content from presentation, making websites more attractive and easier to maintain. CSS works along with HTML and JavaScript to create visually appealing, user-friendly, and responsive web interfaces across different devices and screen sizes.
JavaScript
JavaScript is a powerful scripting language used to create dynamic and interactive web pages by enabling features such as form validation, animations, event handling, and real-time content updates. It works alongside HTML and CSS to enhance user experience, supports both client-side and server-side development, and plays a key role in building modern, responsive web applications.
React
React is a popular JavaScript library used to build fast and interactive user interfaces, especially for single-page applications. It uses a component-based architecture and a virtual DOM to efficiently update and render UI elements, making applications scalable, reusable, and easy to maintain.
